# CLAUDE.md

## Rules

- **Never read or view image/photo files** (jpg, jpeg, png, webp, gif, heic, etc.) using the Read tool or any other tool that would display their visual contents. You may move, rename, copy, list, and read metadata (e.g. exiftool, stat) of image files, but never view them.
- **Never read or view video files** (mp4, mov, avi, mkv, webm, etc.) using the Read tool or any other tool. You may move, rename, copy, list, and read metadata of video files, but never view them.
